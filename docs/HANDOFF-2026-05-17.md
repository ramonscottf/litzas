## Design direction to preserve

- **Colors locked:** `#ae9860` gold, warm black, warm off-white. Nothing else.
  Depth comes from glass, shadow, and transparency — not from new hues.
- **Fonts locked:** Anton (LITZAS hero), Alfa Slab One (PIZZA slab), Oswald
  (labels/eyebrows), Inter (body).
- **No entrance animations on hero sections.** Functional micro-motion only.
- **Family is real, not aesthetic.** Don Hale → Hires (1959) → Litzas (1965).
  Same family today. Don't write "sister brand bridge" copy — write "same
  company" copy.
- **Underdog tone:** 1980s neighborhood pizzeria. Not a date-night place,
  not a reservation place. Walk in, eat pizza, leave. Fresh, hot, classic.

## Photo phase

The public menu only shows approved pizza photos. Pizzas without approved
photos render as text-first cards with a quiet "Photo in approval" badge.

Source of truth: `data/menu-photo-manifest.json`. For each pizza:

1. Place the approved image at `assets/images/menu/pizzas/<slug>.jpg`.
2. Update `approvalStatus` to `approved-existing` or `approved-generated`.
3. Keep `spotonExportName` aligned.
4. Re-render and rebuild:

```bash
node scripts/render-site.mjs
node scripts/package-spoton-photos.mjs
npm test
npm run check
```
